file  cutting_distance_solver.h [code]
 Given an orientation of the cutting plane, uses secant-bisection algorithm to solve for the cutting distance that matches the target volume fraction to the given tolerance. All the provided material polygons are cut with the same plane. The unique position of the plane is determined based on the aggregate moments of the components in the lower half-space (below the plane).

file  nested_dissections.h [code]
 Implements the nested dissections algorithm. Each instance of this class is associated with a specific CellID. If is possible to sequentially use the same instance of this class for different mesh cells because cellID is a reference to the variable that can be modified externally. Materials order should generally be updated if cellID is modified.
