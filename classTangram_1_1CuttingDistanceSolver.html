Detailed Description

template<int D, class MatPoly_Clipper>
class Tangram::CuttingDistanceSolver< D, MatPoly_Clipper >

For a vector of MatPoly's and orientation of the cutting plane, finds the position of the plane for which the aggregated moments of their parts in the lower half-space match the specified volume fraction to the given tolerance.

By default, the faces of MatPoly's are assumed to be planar and are NOT facetized. If planar_faces is set to false, faces of all MatPoly's are assumed to be non-planar and are facetized.
